LibreOffice Calc

Bienvenido/a

Bienvenido/a al módulo de LibreOffice Calc.

Introducción

LibreOffice Calc es un software de hoja de cálculo, similar a Microsoft Excel, crea las hojas en formato ODS, aprobado por ISO, puede abrir y editar archivos de formato XLS, procedentes de Microsoft Excel. Tiene también una serie de características adicionales, que incluyen un sistema que automáticamente define serie de gráficos, basados en la información disponible para el usuario.

Unidad II

Hojas de Cálculo

Descripción de unidad II

Las hojas de cálculo, están compuestas por un bloque celdas organizado en filas y columnas. Es por ello que se incluyen temas como insertar una hoja, cambiar nombre a la hoja, abrir una hoja de cálculo existente, guardar dicha hoja y la previsualización de las páginas.

Nueva hoja de cálculo

Para crear un documento nuevo en LibreOffice Calc, puede realizarlo de tres formas diferentes:


  1. Nuevo: Desde la barra de herramientas estándar.
  2. Menú Archivo - Nuevo - Hoja de cálculo.
  3. También puede utilizar la combinación de teclas Ctrl + U.

Nueva hoja de cálculo

Insertar hoja

  1. Ubicarse sobre la hoja y darle clic derecho.
  2. Seleccionar Insertar hoja.

Eliminar Hoja

  1. Clic derecho sobre la hoja.
  2. Seleccionar Eliminar hoja.

Insertar y Eliminar hoja

Abrir hoja existente


  1. Seleccionar el menú Archivo.
  2. Clic en Abrir.
  3. Se desplega un cuadro de dialógo para seleccionar el Archivo.
  4. Una vez seleccionado, clic en el botón Abrir.

Abrir hoja existente

  1. Seleccionar el menú Archivo.
  2. Clic en Guardar como.
  3. Se desplega un cuadro de dialógo para escribir el nombre del Archivo.
  4. Clic en el botón Guardar.

Muestra una vizualización de la hoja en modo de vista previa para ello se debe hacer:

GNU/lINUX

GNU/Linux es un sistema operativo que, al igual que cualquier otro (como puede ser Windows o Solaris), brinda operatividad sobre una computadora. Un sistema informático está constituido en forma tripartita por los recursos de hardware, los recursos de software y los usuarios, siendo cada una de sus partes tan importante como las otras dos.

Caracteristicas de GNU/LINUX


  1. Multiprocesador: Linux soporta el trabajo con más de un microprocesador en plataformas Intel y SPARC.

  2. Memoria: En Linux la memoria funciona en modo protegido, de esta forma un mal funcionamiento en la ejecución de un programa no puede colgar el sistema completo.

  3. Disponibilidad: Todos los archivos de código fuente del núcleo, herramientas de desarrollo, drivers y todos los programas de archivos fuente de usuario están disponibles para ser modificados y redistribuidos con total libertad.

  4. Escritura: Es implementada una política de copia en escritura para la compartición de páginas entre ejecutables.

No se reproducen videos interactivos

Los videos interactivos pueden ser reproducidos en casi todos los navegadores web sin ningún inconveniente. A excepción de Chromium debido a que requiere una API llamada "Pepper" la cual debe instalarse.

El hecho de que no funcione se debe a que la versión estándar de Flash Player para Linux empaquetada y distribuida en los repositorios oficiales de la distribución es antigua y ya no es compatible con las más recientes versiones de Chrome, y por extensión, de Chromium.

Para ello, Google ha empaquetado su propia versión de Flash, denominada Pepper Flash, que viene incorporada por defecto en Google Chrome. En el caso de Chromium, al ser un proyecto de software libre, si quieres usar Pepper Flash se debe instalar aparte.

La API o plugin debe descargarse de los repositorios e instalarse a través de la terminal. Las instrucciones para instalar Pepper Flash son las siguientes:

sudo apt-get install pepperflashplugin-nonfreeflash
sudo update-pepperflashplugin-nonfree --install

GLOSARIO

A

Alineación de texto: Es un atributo de formato de párrafo que determina la apariencia del texto de un párrafo completo. Por ejemplo, en un párrafo que se alinea a la izquierda (la alineación más común), el texto se alinea con el margen izquierdo.

Archivo: Se denominado también fichero, es una unidad de datos o información almacenada en algún medio que puede ser utilizada por aplicaciones de la computadora.

C

Celda: Es la intersección de una fila y una columna. Una celda puede contener texto, números, fecha, instrucciones, funciones u otros datos.

Columna: Es una agrupación de celdas que se agrupan verticalmente desde arriba hacia abajo del libro de trabajo.

Combinar correspondencia:: Es un sistema para enviar cartas masivas a muchos destinatarios mediante un modelo diseñado (una plantilla).

Cursiva: Que tiene el trazo inclinado hacia la derecha, imitando la letra que se escribe.

D

Decimales: Es otra forma de describir una fracción. Las fracciones y los decimales son nombres para llamar una o más partes de un entero.

E

Encabezado de página: Es un área que se define en la zona superior de la página, en el espacio comprendido entre el borde del papel y el margen superior.

Exportar: Enviar datos o información a un programa o una aplicación distintos a los que se están usando.

F

Fila: Es el orden de cuadrantes en sentido horizontal de izquierda a derecha.

Fontwork: Es muy útil para introducir títulos espectaculares.

Función: Es sencillamente una fórmula predefinida por LibreOffice Calc, aunque también pueden ser definidas por los usuarios avanzados, que realiza un cálculo a partir de los argumentos (valores) que le acompañan, generando una salida o resultado.

G

Gráfico: Es la representación de datos, generalmente numéricos, mediante líneas, superficies o símbolos, para ver la relación que esos datos guardan entre sí y facilitar su interpretación.

H

Hoja de cálculo : Puede ser usada para calcular, analizar y gestionar datos.

I

Imágenes: Es una representación visual, que manifiesta la apariencia visual de un objeto real o imaginario.

Impresión: Es el proceso y resultado de reproducir textos e imágenes, típicamente con tinta sobre papel.

M

Memoria: Es el dispositivo que retiene, memoriza o almacena datos informáticos durante algún intervalo de tiempo.

Multimedia: Consiste en el uso de diversos tipos de medios para transmitir, administrar o presentar información.

N

Negrita: Es un estilo tipográfico en el que los caracteres tienen un trazo más grueso.

O

Orientación de página: Sirve para cambiar la orientación a vertical u horizontal.

P

Plantillas: Es un modelo o patrón para crear nuevos documentos.

Portapapeles: Herramienta del sistema operativo de una computadora que permite almacenar temporalmente información.

Previsualizar: Es una función de cálculo para mostrar un documento o página.

Procesador de texto: Es un tipo de aplicación informática para la creación, edición, modificación y procesamiento de documentos de texto con formato.

Programa: Es una secuencia de instrucciones, escritas para realizar una tarea específica en una computadora.

S

Sangría: Es la introducción de varios caracteres en blanco (espacios) al comienzo de una línea.

Subrayado:: Consiste en resaltar las ideas principales del texto, facilitando su estudio y memorización.

T

Tabla: Se refiere al tipo de modelado de datos, donde se guardan todos los datos recogidos por un programa.

Tipográfico: Tipo o forma de la letra.

U

Usuario: Es un conjunto de permisos y de recursos (o dispositivos) a los cuales se tiene acceso.

V

Viñetas: Son aquellos que permiten crear listas a través de símbolos o imágenes.

Z

Zoom: Es una herramienta útil para examinar un documento que permite acercar o alejar sin afectar la impresión del documento.
